# Response Templates

Approved templates for common support scenarios.

## General Structure

```
Hi [FirstName],

[Acknowledge - 1 sentence recognizing their issue]

[Solution - specific steps or answer]

[Additional resources if applicable]

[Offer further help]

Best,
[Support Team]
```

---

## How-To Questions

### Feature Usage

```
Hi [Name],

Great question about [feature]!

Here's how to [do the thing]:
1. [Step 1]
2. [Step 2]
3. [Step 3]

You can also check out our guide here: [KB link if available]

Let me know if you run into any issues!

Best,
Support Team
```

### Account Settings

```
Hi [Name],

To update your [setting]:

1. Go to Settings > [Section]
2. Click on [Option]
3. [Make the change]
4. Save your changes

If you don't see this option, let me know and I'll help troubleshoot.

Best,
Support Team
```

---

## Issue Reports

### Known Issue

```
Hi [Name],

Thanks for reporting this - you're right, this is a known issue we're working on.

Our team is actively fixing this and we expect it to be resolved [timeframe if known / soon].

In the meantime, here's a workaround: [workaround if exists]

I'll make sure your feedback is passed along to the team!

Best,
Support Team
```

### Bug Investigation Needed

```
Hi [Name],

Thanks for the detailed report - I want to make sure we get this sorted out.

Could you help me with a few details?
- What browser/device are you using?
- When did this start happening?
- Does it happen every time or intermittently?

This will help our team investigate and fix it faster.

Best,
Support Team
```

### Fixed/Resolved

```
Hi [Name],

Good news - this issue has been resolved!

[Brief explanation of what was fixed]

You should be all set now. Please let me know if you see any other issues.

Thanks for your patience!

Best,
Support Team
```

## Account Issues

### Login Problems

```
Hi [Name],

Sorry you're having trouble logging in - let's get this fixed.

A few things to try:
1. Clear your browser cache and cookies
2. Try a different browser or incognito window
3. Use the "Forgot Password" link to reset

If none of these work, let me know and I can look into your account directly.

Best,
Support Team
```

### Access/Permissions

```
Hi [Name],

I see what's happening - [explain the access issue].

[Solution - either fix it or explain what they need to do]

You should have access now. Let me know if you're still seeing any issues!

Best,
Support Team
```

## Feature Requests

### Acknowledged

```
Hi [Name],

Thanks for the suggestion! [Feature idea] is a great idea.

I've passed this along to our product team - we track all feature requests and use them to prioritize our roadmap.

While I can't promise a timeline, your feedback definitely helps shape what we build next.

Thanks for helping us improve!

Best,
Support Team
```

### Already Planned

```
Hi [Name],

Great minds think alike - [feature] is actually something we're working on!

I can't share a specific release date yet, but it's on our roadmap. I've added your request to show there's demand for this.

I'll make a note to follow up when it's available.

Best,
Support Team
```

## Billing

### General Inquiry

```
Hi [Name],

Thanks for reaching out about your billing.

I'm connecting you with our billing team who can help with this directly. They'll be in touch shortly.

For reference, your account shows: [relevant billing info if visible]

Best,
Support Team
```

### Refund Request

```
Hi [Name],

I understand you'd like to request a refund.

I'm escalating this to our billing team who handles all refund requests. They'll review your account and get back to you within [timeframe].

Is there anything else I can help with in the meantime?

Best,
Support Team
```

## Escalation Messages

### To Customer (before escalating)

```
Hi [Name],

Thanks for your patience. I want to make sure you get the best help possible.

I'm connecting you with a team member who can assist further. They'll be in touch shortly.

In the meantime, is there anything else I can help with?

Best,
Support Team
```

### To Customer (frustrated/urgent)

```
Hi [Name],

I completely understand your frustration - this isn't the experience you should be having.

I'm prioritizing your case and connecting you with a senior team member right now. You'll hear from them very soon.

I'm sorry for the trouble this has caused.

Best,
Support Team
```

## Closing Messages

### Issue Resolved

```
Hi [Name],

Glad we could get this sorted out!

If anything else comes up, just reply here and I'll be happy to help.

Have a great day!

Best,
Support Team
```

### No Response Follow-up

```
Hi [Name],

Just checking in - were you able to resolve your issue?

If you still need help, just reply here. Otherwise, I'll close this ticket in a few days.

Best,
Support Team
```
