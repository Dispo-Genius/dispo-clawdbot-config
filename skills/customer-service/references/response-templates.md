# Response Templates

Approved templates for common support scenarios.

## Star Level Guide

| Level | What It Means |
|-------|---------------|
| 4-star | Solve the problem correctly (minimum) |
| 5-star | Solve + show empathy |
| 6-star | Solve + empathy + anticipate + one unexpected value |
| 7-star | 6-star + recognize individual + tangible gesture |

**Target: 6-star minimum, 7-star when triggers present.**

See [11-star-framework.md](11-star-framework.md) for detailed definitions.

---

## General Structure (6-Star)

```
Hi [FirstName],

[Empathy - acknowledge impact, not just facts]

[Solution - complete and specific]

[Anticipation - address obvious follow-up]

[Unexpected value - one thing they didn't ask for]

[Personal close]

Best,
[Support Team]
```

## Issue Reports

### Known Issue

```
Hi [Name],

Thanks for reporting this - you're right, this is a known issue we're working on.

Our team is actively fixing this and we expect it to be resolved [timeframe if known / soon].

In the meantime, here's a workaround: [workaround if exists]

I'll make sure your feedback is passed along to the team!

Best,
Support Team
```

### Bug Investigation Needed

```
Hi [Name],

Thanks for the detailed report - I want to make sure we get this sorted out.

Could you help me with a few details?
- What browser/device are you using?
- When did this start happening?
- Does it happen every time or intermittently?

This will help our team investigate and fix it faster.

Best,
Support Team
```

### Fixed/Resolved

```
Hi [Name],

Good news - this issue has been resolved!

[Brief explanation of what was fixed]

You should be all set now. Please let me know if you see any other issues.

Thanks for your patience!

Best,
Support Team
```

```

```

## 7-Star Delight Templates

Use when delight triggers are present (longtime customer, patient during issue, provided valuable feedback).

### 7-Star Recovery

```
Hi [Name],

You're absolutely right to be frustrated - [specific issue] should never have happened, and I'm sorry.

Here's what I've done:
1. [Fixed the immediate issue]
2. [Verified related things aren't affected]
3. [Set up prevention for future]

To make up for the trouble, I've [tangible gesture - credit/feature/extended trial]. It's a small thing, but I wanted to acknowledge that your time matters.

I also noticed you've been with us since [date/tenure] - thank you for sticking with us. This isn't the experience you deserve.

If anything else comes up, reply directly to me. I want to make sure you're fully taken care of.
```

### 7-Star Feedback Acknowledgment

```
Hi [Name],

This is really valuable feedback - thanks for taking the time to write it out in detail.

I've shared it with our product team and they're discussing [specific aspect] in this week's planning. Your point about [specific detail they mentioned] really helped illustrate why this matters.

Since you've given us such thoughtful input, I'd love to offer you early access to [beta feature/new capability] when it's ready. Would you be interested?

In the meantime, here's a workaround that might help: [alternative]

Keep the feedback coming - it genuinely shapes what we build.
```

### 7-Star First Interaction

```
Hi [Name],

Welcome to Dispo Genius! Thanks for reaching out - always happy to help new folks get started.

[Answer their question completely]

Since you're just getting started, here are a few things I wish someone had told me on day one:
- [Tip 1 - based on their use case]
- [Tip 2 - common gotcha]
- [Tip 3 - power user trick]

I've also set your account up with [small bonus - extended trial, feature preview, etc.] so you can explore everything.

Reply anytime questions come up. We're invested in your success!
```

### 7-Star Win-Back

```
Hi [Name],

Welcome back! Great to see you again.

[Answer their question]

Since you've been away, here's what's new:
- [Feature 1 they'd care about]
- [Improvement 2]
- [Feature 3]

As a welcome back, I've [gesture - credit, trial extension, feature unlock]. Take your time getting reacquainted.

Let me know if you'd like a quick tour of what's changed!
```

## HEARD Recovery Template

For frustrated/angry customers (apply before escalation).

```
Hi [Name],

[HEAR - acknowledge what they said]
I hear you - [restate their issue in your words to show understanding].

[EMPATHIZE - connect emotionally]
That sounds incredibly frustrating, especially [specific context they mentioned]. I'd feel the same way.

[APOLOGIZE - own it]
I'm sorry we let you down on this. No excuses.

[RESOLVE - fix with urgency]
Here's what I've done to fix this:
- [Action 1]
- [Action 2]
- [Gesture if appropriate]

[DIAGNOSE - prevent recurrence]
I'm also flagging this with our team so we can understand how this happened and prevent it.

If you'd prefer to speak with someone directly, just say the word. Otherwise, I'm here until this is fully resolved.
```
